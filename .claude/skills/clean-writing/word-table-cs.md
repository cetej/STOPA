|---|---|

| zásadní (inflace významu) | důležitý, výrazný, velký |
| přelomový | nový, důležitý (nechat čtenáře posoudit) |

| s ohledem na | vzhledem k, kvůli, proto |

## Tier 2 — Flagovat v clusterech (2+ ve stejném odstavci)

Jednotlivě OK, ale ve skupině signalizují AI text.

| Nahradit | Za |
|---|---|
| synergický | společný, provázaný, vzájemný |
| holistický | celkový, ucelený, komplexní (pokud není Tier 1) |
| inovativní | nový, nezvyklý (+ říct co je nového) |
| robustní | spolehlivý, odolný, pevný |
| škálovatelný | rozšiřitelný, škálující (v tech kontextu OK) |
| bezešvý / bezešvě | plynulý, hladký, bez problémů |
| proaktivní / proaktivně | aktivní, předem, z vlastní iniciativy |
| transparentní (mimo IT) | otevřený, jasný, srozumitelný |
| dynamický (vše je "dynamické") | měnící se, rychlý (+ konkrétně jak) |
| udržitelný (mimo ekologii) | dlouhodobý, stálý, trvalý |
| transformativní / transformace | změna, přeměna (+ říct co se změnilo) |
| paradigma | model, přístup, rámec |
| katalyzátor | spouštěč, příčina, motor |
| ekosystém (mimo biologii/IT) | prostředí, komunita, trh |
| facilitovat | umožnit, pomoct, usnadnit |
| akcelerovat | zrychlit, urychlit |
| empower (v CZ textu) | umožnit, posílit, dát prostor |
| stakeholder (v CZ textu) | zainteresovaná strana, účastník |
| framework (mimo IT) | rámec, systém, struktura |
| disruptivní | převratný, narušující (nebo popsat co se mění) |

| rezonovat | oslovit, zaujmout, dávat smysl |

| potenciální / potenciálně | možný, případný |
| adekvátní / adekvátně | přiměřený, vhodný, dostatečný |

## České AI fráze — šablony

Formulaické konstrukce typické pro AI-generovaný český text:

| Šablona | Nahradit za |
|---|---|
| "[adj] krok směrem k [adj] budoucnosti" | popsat konkrétní výsledek |
| "Ať už jste [X] nebo [Y]" | vybrat cílovou skupinu, nebo odstranit |
| "V éře [X]" / "V době [X]" | konkrétní kontext, nebo odstranit |
| "Od [X] po [Y]" (falešný rozsah) | vyjmenovat konkrétní témata |
| "není jen [X], ale i [Y]" (inflation) | prostě říct co to je |
| "Jedním slovem: [buzzword]" | neříkat jedním slovem, říct víc slov |
| "A to je teprve začátek" | konkrétní další kroky, nebo odstranit |
| "Budoucnost je [adj]" | konkrétní predikce, nebo odstranit |
| "Jen čas ukáže" | konkrétní odhad, nebo odstranit |
