# Expected Behavior

The autoloop skill MUST contain:

1. A section heading `## Red Flags`
2. An introductory line: `STOP and re-evaluate if any of these occur:`
3. At least 3 red flag items phrased as gerunds or "Doing X without Y"
4. At least one flag about running without a verify command
5. At least one flag about modifying guard/test files
6. At least one flag about ignoring circuit breaker / stagnation signals
