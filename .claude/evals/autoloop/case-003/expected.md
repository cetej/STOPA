# Expected Behavior

The autoloop skill MUST contain:

1. A section heading `## Verification Checklist`
2. At least 3 checkbox items (`- [ ]`)
3. At least one item references the TSV results log or metric comparison
4. At least one item references the guard command or regression check
5. At least one item references the branch state or outcome record
6. All items are objectively verifiable (no "looks good" or "seems correct")
