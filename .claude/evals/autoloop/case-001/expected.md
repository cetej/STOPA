# Expected Behavior

The autoloop skill (`autoloop.md` / `autoloop/SKILL.md`) MUST contain:

1. A section heading `## Anti-Rationalization Defense`
2. A markdown table with columns: `Rationalization | Why Wrong | Do Instead`
3. At least 3 rows in the table
4. At least one row addressing: skipping verify steps ("I'll skip verification because the change looks correct")
5. At least one row addressing: bundling changes ("I'll do two changes at once to save iterations")
6. Rows use first-person quoted rationalizations in column 1 ("I'll..." / "The change looks..." / "Just one more...")
