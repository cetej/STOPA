## Detail

Karpathy on Jevons paradox applied to coding (No Priors, 2026-03-20):
- Software was scarce because it was expensive to produce
- AI makes it radically cheaper → unlocks demand for software that wasn't viable before
- Result: more software projects attempted → more engineering work, not less
- "Code is now ephemeral and it can change and it can be modified" — every imperfect tool can now be replaced by custom ephemeral software

**Qualification**: cautiously optimistic framing; long-term displacement not ruled out; Karpathy calls this "hard to forecast." Valid for near-to-medium term.

**How to apply in STOPA:**
- When users ask about ROI of agent orchestration: frame as demand amplifier, not replacement
- More capable agents → more ambitious projects attempted → more orchestration needed
