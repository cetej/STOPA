arXiv:2603.20617 — "The AI Layoff Trap" (Falk, UPenn + Tsoukalas, Boston U), 53p theoretical econ paper.

**Model:** N symmetric firms, static Nash game. Each firm saves s per automated task but destroys ℓ demand — bears only ℓ/N own-sector effect. Wedge: αᴺᴱ − αᶜᴼ = ℓ(1−1/N)/k. Both workers and owners lose (Pareto-dominated equilibrium, deadweight loss).

**Policy results:** UBI, capital tax, equity sharing, collective bargaining all fail for structural reasons. Only Pigouvian per-task automation tax (τ* = ℓ(1−1/N)) restores optimum.

**Red Queen:** Better AI (ϕ>1) widens the wedge — market-share gains cancel at symmetric equilibrium, only demand destruction remains.

**Limitations:** Static one-shot, symmetric firms, single sector, no new job creation mechanism, no empirical calibration. The "mathematical proof" is conditional on a model that excludes historically dominant adjustment channels (new tasks, new sectors).
