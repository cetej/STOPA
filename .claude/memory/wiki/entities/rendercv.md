# RenderCV

> CLI/YAML-based resume builder pro engineers a akademiky — YAML input s Pydantic validací → Typst typesetter → PDF. V2.0 přešel z LaTeX na Typst kvůli pip-installability a CJK podpoře.

## Key Facts

- GitHub: rendercv/rendercv, 16.2k stars, Python (82.8%) + Typst (16.5%), last commit March 2026 (ref: sources/magic-resume-research-2.md)
- Input: YAML s Pydantic validací + JSON Schema autocomplete (ref: sources/magic-resume-research-2.md)
- v2.0 switch LaTeX → Typst: LaTeX nešel pip-install čistě a chyběla CJK jazyková podpora (ref: sources/magic-resume-research-2.md)
- 9 built-in themes, customizable margins/fonts/colors přes Typst template files (ref: sources/magic-resume-research-2.md)
- Web app: AI career assistant s GitHub sync a YAML+form editorem (rendercv.com) (ref: sources/magic-resume-research-2.md)

## Relevance to STOPA

Dokumentační vzor: YAML-first s Pydantic validation je analogie STOPA SKILL.md YAML frontmatter. Typst jako alternativa k LaTeX pro dokumentaci je relevantní pro STOPA/ORAKULUM report generation.
