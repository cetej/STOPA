# ForecastBench

> Dynamic benchmark for evaluating LLM forecasting ability against human forecasters; projects human-AI parity by November 2026 via linear extrapolation.

## Key Facts

- Paper: arXiv:2409.19839 (Karger et al. 2024) (ref: sources/prediction-research.md)
- Live leaderboard: https://www.forecastbench.org/ (ref: sources/prediction-research.md)
- Parity projection: November 2026 (single-source, linear extrapolation) (ref: sources/prediction-research.md)
- Best LLM (o3) Brier 0.1352 vs superforecaster median 0.0225 — 6× gap as of 2025 (ref: sources/prediction-research.md)

## Relevance to STOPA

Benchmark for evaluating POLYBOT's prediction capabilities. Track ForecastBench leaderboard when selecting LLM for prediction tasks in MONITOR or ZÁCHVĚV.
