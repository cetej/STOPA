# Human-AI Centauri

> Kompozitní lidsko-AI aktori — "ani čistě lidští, ani čistě strojový" — jeden člověk řídí více AI agentů, nebo jedna AI slouží více lidem v měnících se konfiguracích.

## Key Facts

- Koncept z arXiv:2603.20639 (Evans, Bratton, Agüera y Arcas)
- Pokročilá kapabilita: agenti mohou spawnovat interní kopie, diferenciovat pro subtasky, rekombinovat výsledky
- Rekurzivní Societies of Thought expandující s náročností úkolu
- STOPA instance: uživatel + Claude Code + sub-agenti jako kompozitní aktor (ref: sources/agentic-ai-intelligence-explosion.md)

## Relevance to STOPA

STOPA workflow (uživatel → orchestrate → sub-agenti → výsledek) je přímá implementace Human-AI Centauri vzoru. Depth limit >2 v STOPA odpovídá rekurzivnímu omezení.
