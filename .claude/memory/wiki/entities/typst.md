# Typst

> Moderní open-source typesetting system jako náhrada za LaTeX — pip-installable, Unicode/CJK podpora, rychlejší kompilace, jednodušší syntaxe.

## Key Facts

- Pip-installable (na rozdíl od LaTeX/PDFLaTeX) — klíčová výhoda pro Python projekty (ref: sources/magic-resume-research-2.md)
- Nativní Unicode a CJK jazyková podpora (LaTeX to bez extensí nezvládá) (ref: sources/magic-resume-research-2.md)
- RenderCV přešel na Typst v2.0 jako primární PDF rendering engine (ref: sources/magic-resume-research-2.md)
- Menší ekosystém než LaTeX, ale rychle rostoucí (ref: sources/magic-resume-research-2.md)

## Relevance to STOPA

Kandidát pro PDF generování v STOPA report/documentation pipeline (alternativa k LaTeX v ORAKULUM). Pip-installability je klíčová pro Python-centric projekty jako ORAKULUM.
