# Sesame CSM-1B

> Open-source conversational TTS model from Sesame AI Labs (1B params, Apache 2.0), built on Llama backbone + Mimi audio decoder with RVQ tokens; optimized for multi-speaker context-aware speech.

## Key Facts

- Released March 13, 2025; Apache 2.0 license (ref: sources/podcast-tts-research.md)
- Architecture: Llama backbone + Mimi audio decoder, RVQ tokens (ref: sources/podcast-tts-research.md)
- CPU and GPU compatible; real-time capable on both (ref: sources/podcast-tts-research.md)
- Blind tests: participants could not distinguish from real humans in short conversational excerpts (ref: sources/podcast-tts-research.md)
- GitHub: github.com/SesameAILabs/csm; roadmap: 20+ language expansion (ref: sources/podcast-tts-research.md)
- Best for: conversational AI / real-time dialogue — not podcast production (ref: sources/podcast-tts-research.md)

## Relevance to STOPA

Candidate for NG-ROBOT media expansion (voice/audio output) or any real-time conversational agent interface. CPU-compatible is significant for deployment without GPU.
