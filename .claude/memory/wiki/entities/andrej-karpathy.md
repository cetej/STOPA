# Andrej Karpathy

> AI researcher and educator; ex-OpenAI founding team, ex-Tesla Autopilot; creator of nanoGPT, microGPT, and the auto-research concept; advocate for removing humans from optimization loops.

## Key Facts

- Shifted from 80% manual / 20% agent coding to 2% manual / 98%+ agent by December 2025 — "I haven't typed a line of code since December" (ref: sources/karpathy-nopriors-autoagent-loopy-era.md)
- Coined "AI psychosis": state of being overwhelmed by the rate of agent capability unlocks, feeling behind if subscription tokens go unused (ref: sources/karpathy-nopriors-autoagent-loopy-era.md)
- Created microGPT: 200-line Python GPT training implementation — distilled LLM training to minimal essence (ref: sources/karpathy-nopriors-autoagent-loopy-era.md)
- Built "Dobby the Elf Claw": home automation claw controlling Sonos, lights, HVAC, shades, pool, spa, security camera via WhatsApp (ref: sources/karpathy-nopriors-autoagent-loopy-era.md)
- Core philosophy: "name of the game is to increase leverage — few tokens in, huge output on your behalf" (ref: sources/karpathy-nopriors-autoagent-loopy-era.md)
- Published Karpathy Wiki Pattern (~April 2, 2026): LLM-maintained KB architecture (raw/ → wiki/ → schema) (ref: sources/karpathy-wiki-pattern / swarm-kb-research.md)

## Relevance to STOPA

Karpathy's framing of orchestration-as-leverage directly validates STOPA's design goals. His auto-research loop concept is the theoretical basis for STOPA's `/autoresearch` and `/autoloop` skills. The "program.md" idea (research org as markdown files) is the meta-level of what STOPA already does with SKILL.md.
