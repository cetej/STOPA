# GIFT-Eval

> Standard benchmark for time series foundation models: 23 datasets, 144K series, 177M datapoints. Chronos-2 ranks #1 as of 2026.

## Key Facts

- Paper: arXiv:2410.10393 (Aksu et al.) (ref: sources/prediction-research.md)
- 23 datasets, 144K series, 177M datapoints (ref: sources/prediction-research.md)
- Current #1: Chronos-2 (Amazon, 2026) (ref: sources/prediction-research.md)
- Separate from Monash benchmark — TimesFM 2.0 leads Monash (ref: sources/prediction-research.md)

## Relevance to STOPA

Use GIFT-Eval criteria when benchmarking numeric time series components in ORAKULUM or MONITOR prediction pipelines.
