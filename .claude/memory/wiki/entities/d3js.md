# D3.js

> JavaScript knihovna pro datové vizualizace v prohlížeči — force-directed grafy pro vizualizaci znalostních grafů.

## Key Facts

- Použití v STOPA kontextu: force-directed layout pro concept-graph.json vizualizaci (ref: sources/second-brain-gap-analysis.md)
- Alternativa: Cytoscape.js (podobné schopnosti pro grafové vizualizace) (ref: sources/second-brain-gap-analysis.md)
- Implementace: ~300 řádků HTML/JS template + 50 řádků Python generátor pro export z concept-graph.json (ref: sources/second-brain-gap-analysis.md)

## Relevance to STOPA

Technická závislost pro Knowledge Graph Visualization (Cesta B). Žádné externí API, vše běží offline v browseru.
