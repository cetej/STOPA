# DeriNet

> Czech derivational network with 1,040,126 lexemes and 791,771 derivational relations, published by ÚFAL Charles University.

## Key Facts

- DeriNet 2.3: 1,040,126 lexemes with derivational + compound relations (ref: sources/czech-lexical-resources-research.md)
- License: CC BY-NC-SA 4.0 — non-commercial only (ref: sources/czech-lexical-resources-research.md)
- Released January 2025; hosted at LINDAT `hdl.handle.net/11234/1-5846` (ref: sources/czech-lexical-resources-research.md)
- Captures word formation chains e.g. "číst → čtenář → čtenářský" (ref: sources/czech-lexical-resources-research.md)
- No definitions — derivational relationships only (ref: sources/czech-lexical-resources-research.md)

## Relevance to STOPA

Useful for ngm-terminology to find morphological families and generate related term lookups. Enables "find all forms derived from root" queries.
