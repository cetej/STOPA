# kaikki.org

> Parsed Wiktionary data extraction service providing structured JSONL dumps of words from English Wiktionary, covering 68,766 distinct Czech words.

## Key Facts

- Czech extract: 68,766 words, JSONL format (one object per word with definitions, pronunciations, translations, inflections) (ref: sources/czech-lexical-resources-research.md)
- Download: `cs-extract.jsonl.gz` — 36 MB compressed, 260.6 MB uncompressed (ref: sources/czech-lexical-resources-research.md)
- License: Wiktionary CC BY-SA — **commercially usable** (ref: sources/czech-lexical-resources-research.md)
- Based on 2026-03-25 enwiktionary dump; uses wiktextract tool (ref: sources/czech-lexical-resources-research.md)
- This is Czech words as documented in *English* Wiktionary — not Czech Wiktionary itself (ref: sources/czech-lexical-resources-research.md)

## Relevance to STOPA

The most practically useful **commercially-usable** Czech lexical resource. Suitable for ngm-terminology or Czech corrector as a base word list with definitions.
