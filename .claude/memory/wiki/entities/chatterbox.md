# Chatterbox (Resemble AI)

> MIT-licensed open-source TTS from Resemble AI; first open-source model with emotion exaggeration control, zero-shot voice cloning, and imperceptible neural watermarking (PerTh).

## Key Facts

- MIT license — fully permissive including commercial use (ref: sources/podcast-tts-research.md)
- Variants: Chatterbox (base), Chatterbox Multilingual (23 languages), Chatterbox Turbo (<200ms latency) (ref: sources/podcast-tts-research.md)
- 63.75% evaluators preferred it over ElevenLabs; 1M+ Hugging Face downloads (ref: sources/podcast-tts-research.md)
- PerTh: imperceptible neural watermarking built-in (ref: sources/podcast-tts-research.md)
- GitHub: github.com/resemble-ai/chatterbox (ref: sources/podcast-tts-research.md)
- Caveat: GPU required (no CPU support); English-primary for base model (ref: sources/podcast-tts-research.md)

## Relevance to STOPA

Best choice for commercial self-hosted TTS — MIT license removes all licensing friction. Emotion control makes it suitable for expressive podcast/audio content in NG-ROBOT or future GRAFIK voice layer.
