# arXiv:2603.19138 — Implicit Patterns in LLM-Based Binary Analysis

> První systematická trace-level analýza emergentních reasoning vzorů v LLM binary vulnerability analysis. Identifikuje 4 vzory (P1–P4) v 521 binárních souborech × 4 modelech × 99563 kroků.

## Key Facts

- Autoři: Qiang Li, XiangRui Zhang, Haining Wang (Beijing Jiaotong University / Virginia Tech) (ref: sources/llm-binary-patterns-research.md)
- 4 emergentní vzory: P1 Early Pruning (83.5% sessions), P2 Path Lock-in (97.6%), P3 Targeted Backtracking (93.8%), P4 Knowledge-Guided Prioritization (97.6%) (ref: sources/llm-binary-patterns-research.md)
- P2↔P1 bidirektivní smyčka: 79.4% všech pattern přechodů (ref: sources/llm-binary-patterns-research.md)
- P2 a P4: silně negativně korelované (r = -0.845) — komplementární mechanismy (ref: sources/llm-binary-patterns-research.md)
- Temporální vzory: P2 early-biased (phase 1), P1 mid-biased (phase 6), P3 late-biased (46.5% v phase 10), P4 uniform (ref: sources/llm-binary-patterns-research.md)
- 38.0% sessions (198/521) nalezlo zranitelnost; 306 celkových, CWE-78 dominuje (37.6%) (ref: sources/llm-binary-patterns-research.md)
- Doporučení: "leverage implicit mechanisms rather than override them" — design systémů s emergentními vzory, ne proti nim (ref: sources/llm-binary-patterns-research.md)

## Relevance to STOPA

Přímá mapování na STOPA orchestraci: P2↔P1 smyčka = circuit breaker problem ("3× stejný agent → STOP"). P3 late-phase finding: backtracking/recovery prompting injektovat ke konci session. P4 = LEARNINGS.md semantic routing. Výsledek validuje STOPA pravidlo: circuit breaker je nutný pro přerušení přirozeného P2 lock-in.
