# arXiv:2604.07569 — Learning is Forgetting

> "Learning is Forgetting: LLM Training As Lossy Compression" — Conklin et al. (Princeton + Cohere), ICLR 2026.

## Key Facts

- Authors: Henry C. Conklin, Tom Hosking, Tan Yi-Chern, Julian Gold, Jonathan D. Cohen, Thomas L. Griffiths, Max Bartolo, Seraphina Goldfarb-Tarrant (ref: sources/learning-is-forgetting-llm-lossy-compression.md)
- Main claim: LLM training = lossy compression formalized via Information Bottleneck; first empirical validation at LLM scale (75 models, 6 families) (ref: sources/learning-is-forgetting-llm-lossy-compression.md)
- Key tool: Soft entropy estimator (github.com/hcoxec/soft_h) enabling tractable MI estimation at 32B scale (ref: sources/learning-is-forgetting-llm-lossy-compression.md)
- Top results: Optimality r=0.52, Preference info r=0.76 for benchmark prediction; 2-phase training confirmed; capacity threshold for compression found (ref: sources/learning-is-forgetting-llm-lossy-compression.md)
- Compute: ~3,600 H100 GPU hours; 10-70 min per model on 2-8 H100s

## Relevance to STOPA

Primary reference for IB-based model selection and alignment quality measurement without behavioral evals; theoretical grounding for haiku/sonnet/opus tier scaling limitations.
