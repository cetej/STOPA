# AAIF — Agentic AI Foundation

> Linux Foundation projekt (12/2025) standardizující AGENTS.md a agent interoperability. Platinum members: AWS, Anthropic, OpenAI, Google, Microsoft.

## Key Facts

- Vznik: December 2025, pod záštitou Linux Foundation (ref: sources/idea-file-research.md)
- Standardizuje: AGENTS.md, MCP protokol, goose framework (ref: sources/idea-file-research.md)
- Platinum members: AWS, Anthropic, OpenAI, Google, Microsoft (ref: sources/idea-file-research.md)
- AGENTS.md původně vytvořen OpenAI v August 2025 (ref: sources/idea-file-research.md)
- Relationship ke SKILL.md: paralelní standard — zda splynout nebo zůstanou separátní je nejasné (ref: sources/idea-file-research.md)

## Relevance to STOPA

AAIF governance je relevantní pro STOPA skill distribuce a kompatibilitu. Pokud AGENTS.md a SKILL.md standardy splynout, STOPA bude muset aktualizovat svůj skill format. Sledovat vývoj standardizace.
