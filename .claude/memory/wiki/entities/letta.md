# Letta (formerly MemGPT)

> San Francisco AI company (formerly MemGPT / cpacker) building stateful agent infrastructure; creators of the Letta agent platform and Claude Subconscious plugin.

## Key Facts

- Formerly MemGPT (cpacker GitHub org); rebranded to Letta (ref: sources/claude-subconscious-research.md)
- Builds Letta: persistent agent platform with typed memory blocks, cloud at app.letta.com or self-hosted (ref: sources/claude-subconscious-research.md)
- Released Claude Subconscious (January 2026, MIT license) — Claude Code plugin using Letta as background brain (ref: sources/claude-subconscious-research.md)

## Relevance to STOPA

Letta's architecture (typed memory blocks, per-block token budgets, tool-access modes) is a reference implementation for STOPA's memory system design. The MemGPT concept of virtual context management is the academic precursor to STOPA's checkpoint/memory system.
