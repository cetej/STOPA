# /telescope skill

> Plánovaný STOPA skill (Fáze B) pro explicitní cross-level konzistenci — 3 paralelní analyzéry (mikro/Haiku, mezo/Sonnet, makro/Sonnet) s cross-level synthesis a Vertical Consistency Report.

## Key Facts

- Plánovaný start: 2026-04-21 (Fáze B), po validaci 3-level scout v Fázi A
- 3 paralelní analyzéry: Mikro-Analyst (Haiku), Mezo-Analyst (Sonnet), Makro-Analyst (Sonnet)
- Výstup: Cross-Level Synthesis → Vertical Consistency Report
- Token overhead: +10-20% vs baseline (méně než Fáze C)
- Go/No-Go kritérium: musí zachytit alespoň 1 reálný cross-level problém (ref: sources/vertical-scaling-report.md)

## Relevance to STOPA

Klíčová komponenta Fáze B vertikálního škálování. Rozšiřuje scout o explicitní cross-level validaci. Integruje se do orchestrate po testování.
