# Garry Tan

> President of Y Combinator, author of "Thin Harness, Fat Skills" — the foundational essay on agentic skill architecture.

## Key Facts

- Author of "Thin Harness, Fat Skills" — practical framework for building AI agent systems (ref: sources/garry-tan-thin-harness-fat-skills.md)
- Core thesis: "Fat skills, thin harness, and the discipline to codify everything" is how you get 100× productivity (ref: sources/garry-tan-thin-harness-fat-skills.md)
- Principle: "If I ask you for something twice, you failed" — all repeated tasks must become skills (ref: sources/garry-tan-thin-harness-fat-skills.md)
- Distinguishes latent (model judgment) from deterministic (code) operations as the foundational design decision (ref: sources/garry-tan-thin-harness-fat-skills.md)

## Relevance to STOPA

His three-layer architecture (fat skills → thin harness → deterministic tools) directly maps to STOPA's design: SKILL.md files + hooks layer + Python tools/scripts.
