# How AI Is Transforming Work at Anthropic

**Source**: Anthropic Research  
**Dataset**: 132 survey respondents + 53 interviews + 200K Claude Code transcripts

## Headline Numbers

| Metric | Before | After |
|--------|--------|-------|
| Daily Claude usage | 28% of work | 59% of work |
| Self-reported productivity gain | 20% | 50% |
| Consecutive tool calls without intervention | 9.8 | 21.2 |
| Human interventions per transcript | 6.2 | 4.1 |
| Feature implementation tasks (% usage) | 14% | 37% |
| Design/planning tasks (% usage) | 1% | 10% |

## Delegation Heuristics

Engineers delegate when tasks are:
- **Easily verifiable** — can check output quickly
- **Outside personal context** — no unique knowledge required
- **Low complexity** — well-understood domain
- **Well-defined scope** — clear inputs/outputs
- **"Boring" work** — repetitive, mechanical

**Never delegated**: design decisions, strategic thinking.

## ~27% New Work

Approximately 27% of Claude-assisted work represents entirely new work that wouldn't otherwise occur — not just faster completion of existing work.

## Identity Shift

Engineers describe themselves as "manager of AI agents." Full-stack capability expansion: engineers tackle broader scope with Claude than they would alone.

## Tensions

- Productivity gains ↔ skill erosion concerns (see [ai-coding-skill-atrophy.md](ai-coding-skill-atrophy.md))
- Full-stack expansion ↔ long-term role uncertainty (career anxiety widespread)
- Reduced collaboration friction ↔ lost mentorship/peer-learning opportunities
- Most engineers fully delegate only 0–20% of tasks (most work still requires oversight)

## STOPA Relevance

The delegation heuristics directly apply to STOPA's budget tier decisions: easily verifiable + low complexity + well-defined = `light` tier. Design decisions + strategic thinking = keep in main context. The 21.2 consecutive tool calls data validates STOPA's autonomous execution mode. The 27% new work finding supports the thesis that orchestration systems create value beyond efficiency.
