4-krokový workflow z Building a Second Brain pro transformaci konzumované informace na kreativní výstup.

## CODE

### Capture
Zachyť co rezonuje. Klíčová heuristika: **resonance** — ukládej intuitivně, ne analyticky. Capture by měl trvat sekundy, ne minuty. Nástroje: web clipper, read-later app, highlight exporter.

### Organize
Řaď podle PARA (Projects/Areas/Resources/Archive). Informace žije blízko projektu, který ji využije — ne v tematické hierarchii.

### Distill
Progressive Summarization: highlight → bold → executive summary. Nikdy nemaž originál — přidávej vrstvy.

### Express
Vše směřuje k tvorbě. Knowledge system existuje pro produkci deliverables, ne pro sběratelství.

## Resonance heuristic

"Save what resonates — what feels intuitively important, not what you analytically judge as important." Analytický výběr je pomalý a často špatný. Intuitivní je rychlý a překvapivě přesný pro osobní relevanci.

## Vztah k 2BRAIN 6C cyklu

| CODE | 6C | Mapování |
|------|-----|----------|
| Capture | Capture | Identické — quick entry |
| Organize | Compile + Connect | LLM organizuje automaticky |
| Distill | Compile | LLM kompiluje raw → wiki |
| Express | Consult + Contemplate | Dotazy a reflexe produkují výstupy |

Rozdíl: CODE je manuální workflow. 6C automatizuje Organize a Distill přes LLM compiler.
