Autor knihy "Building a Second Brain" (2022). Zakladatel Forte Labs. Popularizátor osobního knowledge managementu.

## Klíčové příspěvky

### CODE Framework
Capture → Organize → Distill → Express. Systematický workflow pro transformaci informací na kreativní výstup.

### PARA Method
Projects/Areas/Resources/Archive — organizace podle actionability místo tématu.

### Progressive Summarization
Vrstevnatá distilace: highlight → bold → executive summary bez mazání originálu.

### Resonance Heuristic
Ukládej co rezonuje intuitivně, ne co se zdá analyticky důležité.

## Proč je relevantní

Forte's BASB je "human-native" second brain — funguje i bez AI. 2BRAIN staví na BASB principech, ale automatizuje Organize a Distill přes LLM (Karpathyho compiler analogy). Forte poskytuje workflow design, Karpathy technickou implementaci.
