# Nurture-First Agent Development

**Submitted:** March 11, 2026

## Key Claim

Domain expertise is substantially tacit, deeply personal, and continuously evolving. Traditional sequential agent development (design-then-deploy) fails because it treats knowledge encoding as a discrete pre-deployment phase.

## NFD Paradigm

Agents start minimal and evolve through structured dialogue with domain experts. Knowledge is extracted iteratively — not pre-loaded.

## Core Components

1. **Knowledge Crystallization Cycle** — consolidates fragmented operational knowledge into reusable, structured assets
2. **Three-Layer Cognitive Architecture** — organizes knowledge by volatility and personalization level
3. **Dual-Workspace Pattern** — separates exploration from crystallized knowledge
4. **Spiral Development Model** — iterative growth loops

## Case Study

Financial research agent for U.S. equity analysis — demonstrates domain expertise encoding through practitioner interaction.

## Relevance

Bridges PKM (second brain patterns) with agent development methodology. Tacit knowledge crystallization = compiler analogy at the agent level.
