# Andrej Karpathy: Přehled roku 2025 v oblasti velkých jazykových modelů

**URL**: https://karpathy.bearblog.dev/year-in-review-2025/

## Key Idea

Rok 2025 přinesl šest klíčových paradigmatických změn v LLM: RLVR jako novou fázi trénování, pochopení 'zubatého' charakteru AI inteligence, vzestup vertikálních LLM aplikací typu Cursor, AI žijící na lokálním počítači, vibe coding pro všechny a první náznaky GUI pro LLM.

## Claims

- RLVR se stala novou standardní fází tréninku LLM, která umožňuje spontánní vznik reasoning strategií a pohlcuje větší část výpočetního výkonu než pretraining
- LLM inteligence má 'zubatý' charakter - vynikají ve verifikovatelných doménách, ale současně selhávají v základních úlohách, což je odlišuje od lidské nebo živočišné inteligence
- Cursor definoval novou vrstvu LLM aplikací, které orchestrují více LLM volání, provádějí context engineering a nabízejí vertikálně specifické GUI
- Claude Code ukázal správný směr tím, že běží na lokálním počítači uživatele s přístupem k prostředí, datům a nízké latenci namísto v cloudu
- Vibe coding demokratizuje programování - umožňuje nejen běžným lidem tvořit software, ale i profesionálům vytvářet efemérní aplikace pro jednorázové úkoly
- Google Gemini Nano banana naznačuje budoucnost LLM GUI - vizuální a prostorové rozhraní místo pouze textového chatu

## Relevance for STOPA

STOPA orchestrace může využít poznatky o vertikální orchestraci LLM (Cursor model), lokálním běhu agentů s přístupem k prostředí a dat, a principů RLVR pro optimalizaci multi-agentních systémů proti verifikovatelným metrikám.
