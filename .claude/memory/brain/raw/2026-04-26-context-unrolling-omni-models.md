# Context Unrolling v multimodálních modelech Omni

**URL**: http://arxiv.org/abs/2604.21921v1

## Key Idea

Omni je unified multimodální model trénovaný nativně na text, obrázky, videa, 3D geometrii a skryté reprezentace. Model využívá Context Unrolling - explicitní uvažování napříč různými modálními reprezentacemi před generováním predikce.

## Claims

- Context Unrolling umožňuje agregaci komplementárních informací napříč heterogenními modalitami
- Multimodální trénování vede k věrnější aproximaci sdíleného multimodálního knowledge manifoldu
- Omni dosahuje silného výkonu v benchmarkách pro multimodální generování i porozumění
- Model je schopen in-context generování textu, obrázků, videa a 3D geometrie

## Relevance for STOPA

Context Unrolling představuje relevantní přístup pro STOPA orchestraci - schopnost uvažovat napříč různými reprezentacemi před finální predikcí může zlepšit koordinaci mezi heterogenními AI agenty a modalitami v orchestračním frameworku.
