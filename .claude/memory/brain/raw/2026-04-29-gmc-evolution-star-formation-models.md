# Vliv modelů tvorby hvězd na evoluci molekulárních mračen v simulacích galaxií

**URL**: http://arxiv.org/abs/2604.25911v1

## Key Idea

Simulace galaxie podobné NGC 300 ukazují, že životnost obřích molekulárních mračen (GMC) je silně ovlivněna zvoleným modelem tvorby hvězd - gravo-thermo-turbulentní model vytváří mračna s životností >200 Myr, zatímco model se sink částicemi produkuje realistická krátkodobá mračna (~20-30 Myr).

## Claims

- Model se sink částicemi produkuje GMC s životností 20-30 Myr a SFE per free-fall time několik procent, což odpovídá pozorováním
- Gravo-thermo-turbulentní (GTT) model vytváří mračna s extrémně dlouhou životností >200 Myr kvůli velmi nízké SFE (<0.003), což činí stellární feedback neúčinným
- Srážky mračen prodlužují životnost GMC a zvyšují jejich integrovanou SFE prodloužením aktivního období tvorby hvězd, ale mají jen malý dopad na okamžité efektivity
- Oba modely reprodukují pozorovanou Kennicutt-Schmidt relaci s časem vyčerpání plynu několik Gyr
- Extrémní model zpětné vazby (5× energie supernov) nadměrně potlačuje tvorbu hvězd a produkuje velmi dlouhé časy vyčerpání (6-20 Gyr)

## Relevance for STOPA

Demonstruje kritický význam správné volby modelů pro simulaci komplexních fyzikálních systémů - podobně jako orchestrace LLM agentů vyžaduje vhodné modely a strategie pro efektivní koordinaci. Ukazuje trade-off mezi přesností a výpočetní náročností různých přístupů.
