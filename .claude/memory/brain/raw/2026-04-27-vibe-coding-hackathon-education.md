# Vibe Coding Hackathon: Vzdělávací aplikace programování pomocí LLM

**URL**: http://arxiv.org/abs/2604.22747v1

## Key Idea

Studie měsíčního hackathonu zkoumající vzdělávací hodnotu 'vibe codingu' (programování pomocí přirozeného jazyka a AI), kde účastníci všech úrovní vytvářeli projekty pouze pomocí LLM generovaného kódu bez manuálních úprav.

## Claims

- Vibe coding umožňuje programovat pomocí přirozeného jazyka, kdy uživatelé popisují záměr a AI generuje či upravuje kód
- Hackathon nabízel tři úrovně složitosti: Spark (frontend), Build (backend/databáze), Launch (produkční aplikace)
- Účastníci z různých zemí a úrovní dovedností úspěšně vytvářeli projekty bez manuálních úprav kódu, pouze prostřednictvím LLM
- Studie kombinovala standardizované hodnocení projektů s post-hackathon dotazníky a tematickou analýzou zpětné vazby

## Relevance for STOPA

Zkoumá, jak můžou LLM demokratizovat programování a jak různé úrovně zkušeností ovlivňují efektivitu AI-asistovaného vývoje - důležité pro design nástrojů a workflow v STOPA orchestraci.
