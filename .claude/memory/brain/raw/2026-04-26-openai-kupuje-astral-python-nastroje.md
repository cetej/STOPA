# OpenAI kupuje Astral: Python nástroje uv, ruff a ty přecházejí pod křídla AI gigantu

**URL**: https://simonwillison.net/2026/Mar/19/openai-acquiring-astral/

## Key Idea

OpenAI akvizicí Astral získává klíčové Python nástroje (uv, ruff, ty) a talentované Rust vývojáře pro posílení svého Codex týmu. Komunita se obává strategického rizika, když VC-backed firma vlastní kritickou open-source infrastrukturu.

## Claims

- uv byl stažen přes 126 milionů krát měsíčně a stal se jedním z nejpopulárnějších Python nástrojů za pouhé 2 roky
- OpenAI plánuje pokračovat v podpoře open-source nástrojů Astral po dokončení akvizice
- Konkurence mezi Anthropic Claude Code a OpenAI Codex generuje miliardy dolarů ročně z $200/měsíc předplatných
- Astral mělo neoznámené Series A (Accel) a Series B (a16z) investiční kola
- Díky permisivní licenci je možný fork nástrojů jako kredibilní exit strategie v případě problémů

## Relevance for STOPA

Ukazuje trend konsolidace AI a developer tooling. Pro STOPA je relevantní jako příklad strategických akvizic v ekosystému vývojářských nástrojů a rizik závislosti na vendor-controlled open-source infrastruktuře pro orchestraci.
