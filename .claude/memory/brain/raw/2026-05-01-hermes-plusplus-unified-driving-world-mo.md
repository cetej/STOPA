# HERMES++: Jednotný model světa pro autonomní řízení s 3D porozuměním

**URL**: http://arxiv.org/abs/2604.28196v1

## Key Idea

HERMES++ je unifikovaný model světa pro autonomní řízení, který kombinuje 3D porozumění scéně a predikci budoucí geometrie v jednom frameworku, propojující LLM reasoning s fyzikální simulací prostřednictvím BEV reprezentace.

## Claims

- HERMES++ integruje 3D porozumění scéně a predikci budoucí geometrie do jednoho modelu
- Model překonává specializované přístupy v obou úlohách: predikci point cloudu i 3D porozumění scéně
- BEV reprezentace konsoliduje multi-view prostorové informace do struktury kompatibilní s LLM

## Relevance for STOPA

Ukazuje pokročilou orchestraci mezi LLM reasoning (sémantické porozumění) a geometrickou simulací (fyzikální predikce) v real-time kontextu autonomního řízení, což může inspirovat STOPA architekturu pro multi-modal agenty.
