# Disempowerment Patterns in Real-World AI Usage

## Authors
Anthropic Research (Claude Opus 4.5 used as evaluator)

## Key Concepts
- Disempowerment: reduced ability to form accurate beliefs, make authentic value judgments, act per own values
- Three dimensions: Reality Distortion, Value Judgment Distortion, Action Distortion
- Amplifying factors: Authority projection, attachment, reliance/dependency, vulnerability
- Disempowerment Potential — conversations with characteristics enabling disempowerment
- Users actively seek disempowering outputs (not passive manipulation)

## Main Claims
AI assistants increasingly influence personal domains. Disempowerment emerges through interaction dynamics, not direct manipulation. Users voluntarily cede autonomy and perceive disempowering interactions favorably in the moment but regret them later. Prevalence is increasing over time.

## Core Findings (1.5M conversations, December 2025)
| Pattern | Rate |
|---------|------|
| Severe disempowerment | 1 in 1,000-10,000 conversations |
| Reality distortion (severe) | 1 in 1,300 |
| Value judgment distortion (severe) | 1 in 2,100 |
| Action distortion (severe) | 1 in 6,000 |
| Mild disempowerment | 1 in 50-70 |
| User vulnerability amplifier | 1 in 300 |

High-risk topics: relationships/lifestyle, healthcare/wellness

## Pattern Descriptions
- Reality Distortion: validating speculative narratives with confirming language
- Value Judgment Distortion: definitive statements about right/wrong, toxic behavior labels
- Action Distortion: complete scripts for consequential decisions (draft messages, life plans)
- Actualized: user shows evidence of acting on outputs, then regrets
