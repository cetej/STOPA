## Summary

Simon Willison announced first two chapters of "Agentic Engineering Patterns" guide, distinguishing professional SE augmented by coding agents from "vibe coding" (non-programmers using LLMs without expertise). Chapter 1 ("Writing code is cheap now"): reduced code generation costs reshape workflows. Chapter 2 ("Red/green TDD"): test-first development enables agents to produce more succinct, reliable code with minimal extra prompting. Evergreen chapters updated over time, all content written personally by Willison.

## Key Concepts

- Agentic engineering = professional SE + coding agents (not vibe coding)
- Vibe coding = non-programmers using LLMs without engineering expertise
- Code is cheap now: cost reduction reshapes development workflows
- Red/green TDD with agents: test-first enables succinct, reliable AI code
- Minimal prompting with TDD: test structure provides sufficient guidance
