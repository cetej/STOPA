## Summary

Karpathy endorses "Farzapedia" — a personal Wikipedia built by LLM from 2,500 diary entries and messages — as better model for AI personalization than proprietary AI systems. Four advantages: explicitness (see exactly what AI knows), data ownership (files stay on your machine), universal format (plain markdown/images, works with Unix + fine-tuning), and AI provider flexibility (swap providers freely). Managing a personal knowledge wiki = core 21st-century skill AI agents can help maintain.

## Key Concepts

- Farzapedia: personal Wikipedia from 2,500 diary entries (Farza @FarzaTV)
- Explicit memory > implicit AI memory (you can see what it knows)
- Data ownership: your files, not locked in provider
- Universal format: markdown/images → any tool, fine-tuning compatible
- Vendor independence: swap AI providers freely
- LLM wiki as 21st-century life skill
