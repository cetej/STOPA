# Raw Extraction

## Key Concepts
- Memory externalization: state management outside model weights
- Skills externalization: procedural expertise in dedicated stores
- Protocols externalization: interaction structure formalized beyond the model
- Harness engineering: coordination layer unifying all external components
- Cognitive artifacts: framework borrowed from cognitive science

## Main Claims
- Modern LLM agents progress "less by changing model weights than by reorganizing the runtime around them"
- "Agent infrastructure matters" — transforms difficult cognitive tasks into tractable forms
- Three coupled forms of externalization interact; their combination enables emergent behaviors
- Emerging direction: self-evolving harnesses

- Systems-level review (54 pages)
- Historical progression: weights-based → context-aware → harness-based
