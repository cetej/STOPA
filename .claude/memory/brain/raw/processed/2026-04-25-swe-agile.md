# SWE-AGILE

## Authors
Shuquan Lian, Juncheng Liu, Yazhe Chen, Yuhong Chen, Hui Li (KDEGroup)

## Key Concepts
- Dynamic Reasoning Context (DRC) strategy: sliding window of recent reasoning + compressed historical digests
- Tackles dilemma: keep all reasoning (context overflow) vs discard (redundant re-analysis)
- Targeted at 7B-8B small models for SWE tasks
- Prevents Lost-in-the-Middle degradation in long agentic trajectories

## Main Claims
ReAct-style agents lack sophisticated reasoning depth, while extended-thinking models hit context limits in multi-turn tasks. DRC bridges by preserving immediate reasoning verbatim and summarizing older steps.

## Core Findings
- New SOTA for 7B-8B models on SWE-Bench-Verified
- Trained on only 2.2k trajectories / 896 tasks (data-efficient)
- Code: GitHub KDEGroup/SWE-AGILE
