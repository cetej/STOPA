# Raw: Context Awareness Gate (CAG)

## Abstract Summary
RAG degrades when irrelevant context is retrieved — extraneous information impairs LLM's ability to use internal knowledge. CAG dynamically determines whether retrieval is needed per-query.

## Key Contributions
1. Context Awareness Gate (CAG): dynamic mechanism deciding whether external context retrieval is needed
2. Vector Candidates method: statistical, LLM-independent, highly scalable mathematical component
3. Statistical analysis of distributions of context-question relationships

## Main Argument
Improving retrieved chunk quality is insufficient; preventing irrelevant information inclusion is equally critical.

## Concepts
- Adaptive RAG (selective retrieval)
- Context contamination
- Statistical retrieval gating
