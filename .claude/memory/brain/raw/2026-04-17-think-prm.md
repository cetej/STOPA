# ThinkPRM: Process Reward Models That Think (arXiv:2504.16828)

Autoři: Muhammad Khalifa, Rishabh Agarwal, Lajanugen Logeswaran, Jaekyeom Kim, Hao Peng, Moontae Lee, Honglak Lee, Lu Wang.

Problem: Discriminativní PRMs (Process Reward Models) hodnotí kroky přímou klasifikací — bez reasoning. Vyžadují velké množství anotovaných procesních labelů.

Řešení: ThinkPRM — generativní verifier, který produkuje chain-of-thought reasoning při každém kroku verifikace. Místo klasifikace "dobrý/špatný krok" model aktivně reasonuje o správnosti každého kroku.

Klíčový insight: Dlouhé CoT modely mají inherentní reasoning schopnosti — ThinkPRM je využívá pro verifikaci namísto přidávání diskriminativní hlavy. Potřebuje pouze 1% procesních labelů z PRM800K datasetu.

Výsledky:
- ProcessBench, MATH-500, AIME '24: překonává discriminativní verifiery
- GPQA-Diamond: +8% na out-of-domain tasků
- LiveCodeBench: +4.5%
- Vs LLM-as-a-Judge: +7.2% při stejném token budgetu
