# Cenové změny u Claude Code, GitHub Copilot a vydání Qwen3.6-27B

**URL**: https://simonwillison.net/2026/Apr/22/

## Key Idea

Anthropic dočasně změnila ceny Claude Code na 100$/měsíc, GitHub Copilot upravil limity kvůli agentním workflowům a Qwen vydal výkonný open-weight model Qwen3.6-27B s 27B parametry.

## Claims

- Agentní workflowy fundamentálně změnily výpočetní nároky coding asistentů - uživatelé nyní spalují o řád více tokenů než před šesti měsíci
- GitHub Copilot pozastavil registrace pro individuální plány a přešel z per-request na per-token pricing kvůli vysokým nárokům agentních sessionů
- Qwen3.6-27B dosahuje výkonu srovnatelného s předchozím MoE modelem Qwen3.5-397B při zlomkové velikosti (55.6 GB vs 807 GB)
- Claude Mythos Preview identifikoval 271 bezpečnostních zranitelností ve Firefoxu 150, což demonstruje potenciál AI pro defenzivní bezpečnost

## Relevance for STOPA

Ukazuje dramatický nárůst výpočetních nároků agentních orchestrací a ekonomické dopady - důležité pro plánování nákladů STOPA systémů. Qwen3.6-27B nabízí open-weight alternativu s vysokým výkonem pro lokální nasazení.
