# Vibe Coding Hackathon: Vzdělávací přístup k programování přes AI

**URL**: http://arxiv.org/abs/2604.22747v1

## Key Idea

Výzkumná studie měsíčního online hackathonu, kde účastníci všech úrovní vytvářeli projekty pouze pomocí LLM generovaného kódu bez ručních úprav, zkoumající vzdělávací hodnotu 'vibe coding' přístupu k programování pomocí přirozeného jazyka.

## Claims

- Vibe coding umožňuje začátečníkům i pokročilým vývojářům vytvářet funkční aplikace pouze pomocí přirozeného jazyka a LLM
- Omezení 'žádné ruční úpravy kódu' formuje způsob, jak účastníci přistupují k promptování a debuggingu
- Hackathon s třemi úrovněmi obtížnosti (Spark, Build, Launch) demonstruje škálovatelnost AI-asistovaného programování pro různé úrovně dovedností

## Relevance for STOPA

Ilustruje vzdělávací využití AI-asistovaného vývoje a ukazuje, jak strukturované omezení (žádné ruční úpravy) může formovat učební vzorce při práci s LLM v programování, relevantní pro návrh AI-orchestračních workflow.
