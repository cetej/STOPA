# Predikce vývoje LLM a AI kódování pro roky 2026-2032

**URL**: https://simonwillison.net/2026/Jan/8/llm-predictions-for-2026/

## Key Idea

Simon Willison sdílí predikce o dopadu LLM na programování: v roce 2026 bude nesporné, že LLM píší kvalitní kód, do 3 let se vyřeší Jevonsův paradox v softwarovém inženýrství a do 6 let zanikne ruční psaní kódu jako primární činnost.

## Claims

- Do konce roku 2026 nebude možné popírat, že LLM píší kvalitní kód, což poškodí kredibilitu skeptiků
- Rok 2026 přinese konečně řešení sandboxingu pro bezpečné spouštění cizího kódu
- Hrozí 'Challenger disaster' v oblasti bezpečnosti coding agentů, pravděpodobně formou prompt injection wormu
- Do 3 let se vyřeší, zda Jevonsův paradox zachrání kariéry programátorů (vyšší poptávka) nebo je znehodnotí
- Do 3 let někdo vytvoří plnohodnotný webový prohlížeč primárně pomocí AI asistence a nebude to překvapení
- Do 6 let přestane být ruční psaní kódu placená činnost, podobně jako zanikly děrné štítky

## Relevance for STOPA

STOPA orchestrace musí počítat s masivním nárůstem AI-generovaného kódu a coding agentů. Klíčové jsou otázky sandboxingu, bezpečnosti proti prompt injection a potřeba conformance testů pro validaci AI výstupů.
