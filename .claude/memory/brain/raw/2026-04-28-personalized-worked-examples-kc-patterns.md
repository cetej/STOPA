# Personalizované výukové příklady z kódu studentů pomocí KC vzorů

**URL**: http://arxiv.org/abs/2604.24758v1

## Key Idea

Metoda pro automatické generování personalizovaných výukových příkladů z kódu studentů pomocí AST analýzy a vzorů znalostních komponent (KC), které kondicionují generativní model pro lepší pedagogickou relevanci.

## Claims

- KC-kondicionované generování zlepšuje tematické zaměření a relevanci k logickým chybám studentů oproti baseline modelům
- Pattern-based KC extrakce z AST struktury studentského kódu umožňuje identifikovat opakující se strukturální vzory
- Navržený přístup eliminuje nutnost rozsáhlých fixních knihoven výukových příkladů a umožňuje personalizaci na škále

## Relevance for STOPA

Demonstruje využití strukturální analýzy (AST) a pattern extraction pro kondicionování LLM výstupů, což je relevantní pro STOPA orchestraci při generování kontextově přizpůsobeného obsahu z reálných dat uživatelů.
