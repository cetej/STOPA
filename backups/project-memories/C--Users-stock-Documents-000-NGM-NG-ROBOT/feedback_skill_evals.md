Při tvorbě/editaci skills pro generování obrázků:
- Nepoužívat rigidní šablony variant (metaforická koláž / editorial / symbolická) — omezují kreativitu
- Nano Banana Pro přijímá strukturované JSON prompty až 15 000 znaků — limit 30-50 slov je jen pro Midjourney fallback
- Aspect ratio nemá být hardcoded per typ — uživatel může zadat libovolný poměr

**Why:** Uživatel zjistil, že defaultní šablony produkují formulaické výstupy a 30-50 word limit byl zbytečně omezující pro jejich primární generátor (Nano Banana Pro).

**How to apply:** Při práci s vizuálními skills vždy nabízet oba formáty (Nano Banana JSON + Midjourney fallback). Nechat koncepty volné, vycházející z obsahu článku.
